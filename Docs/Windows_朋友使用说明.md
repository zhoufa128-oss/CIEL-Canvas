# Windows 朋友使用说明

1. 下载完整 ZIP，并向开发者核对 SHA-256。
2. 完整解压；不能直接在 ZIP 中运行。
3. 双击 `CIEL Canvas.exe`。
4. 第一次如出现 SmartScreen，先核对 SHA-256、确认文件确由开发者本人提供，再按系统允许方式运行。
5. 不需要安装 Python，也不需要管理员权限。
6. API 需要自行配置；发行包不附带密钥。
7. 所有个人数据都在同目录 `UserData`。
8. 不要删除 `Application` 或 `UserData`。
9. 遇到问题请保留日志并导出诊断资料。

本说明不代表 EXE 已签名，也不保证 SmartScreen 不提示。
